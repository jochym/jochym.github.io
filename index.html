My published projects

  • qe-util a set of utilities for using Quantum-Espresso with ASE and ipython notebooks.
  • qe-doc a set of tutorials for Quantum-Espresso environment inside IPython notebooks developed as a colaborative project between the Institute of Nuclear Physics and the University of Saskachwan.
